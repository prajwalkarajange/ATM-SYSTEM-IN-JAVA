# 🏧 ATM System (Java Project)

A simple **ATM Simulation System** developed using **Java** that allows users to perform basic banking operations such as **checking balance, depositing money, withdrawing cash, and exiting the system securely using a PIN**.

This project demonstrates the use of **Java fundamentals, control structures, loops, and user interaction through the console.**

---

## 📌 Features

- ✅ Secure **PIN Authentication**
- ✅ **Check Account Balance**
- ✅ **Deposit Money** into account
- ✅ **Withdraw Money** with balance validation
- ✅ **User-friendly Console Interface**
- ✅ **Continuous Transaction System** until user exits

## ⚙️ How the System Works

1️⃣ User starts the program.

2️⃣ The system asks the user to **enter the PIN**.

3️⃣ If the PIN is correct, the ATM menu appears:


1: Check Balance <br>
2: Deposit <br>
3: Withdraw <br>
4: Exit <br>


4️⃣ The user selects an option and the system performs the action.

5️⃣ The system continues until the user chooses **Exit**.
